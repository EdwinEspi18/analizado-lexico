Documentacion del Lenguaje

Este documento describe el lenguaje analizado por el compilador: un subconjunto de ECMAScript 2020+ (JavaScript moderno) con soporte completo de lexico, sintaxis y semantica.

Introduccion

El analizador procesa tres fases de compilacion:

  • Lexico — tokeniza el codigo fuente usando un lexer FLEX (C compilado).
  • Sintactico — construye un AST (arbol sintactico) con un parser de descenso recursivo.
  • Semantico — valida alcances, tipos, variables no declaradas, y genera una tabla de simbolos.
  • Traduccion — convierte el AST al lenguaje destino (Python o Java).

Flujo de Control

if / else

if (condicion) {
    // bloque verdadero
} else if (otraCondicion) {
    // ...
} else {
    // bloque falso
}

for (3 formas)

// Clasico
for (let i = 0; i < 10; i++) { ... }

// for-of (valores de iterable)
for (const item of array) { ... }

// for-in (claves de objeto)
for (const clave in objeto) { ... }

while / do-while

while (condicion) { ... }

do {
    // ejecuta al menos una vez
} while (condicion);

switch

switch (valor) {
    case 1:
        console.log("uno");
        break;
    case 2:
        console.log("dos");
        break;
    default:
        console.log("otro");
}

break / continue

for (let i = 0; i < 10; i++) {
    if (i === 3) continue;  // salta esta iteracion
    if (i === 7) break;     // sale del bucle
}

Funciones

Declaracion de funcion

function suma(a, b) {
    return a + b;
}

Expresion de funcion

const multiplicar = function(a, b) {
    return a * b;
};

Arrow function

// expresion concisa (sin return explicito)
const doble = x => x * 2;

// con multiples parametros
const suma = (a, b) => a + b;

// con cuerpo de bloque
const factorial = (n) => {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
};

Parametros especiales

// valor por defecto
function saludar(nombre = "mundo") {
    return `Hola, ${nombre}!`;
}

// rest parameter (variadic)
function sumar(...numeros) {
    return numeros.reduce((a, b) => a + b, 0);
}
Las funciones son ciudadanos de primera clase: pueden asignarse a variables, pasarse como argumentos y devolverse como valores.

Clases

class Animal {
    constructor(nombre, edad) {
        this.nombre = nombre;
        this.edad   = edad;
    }

    saludar() {
        return `Hola, soy ${this.nombre}`;
    }

    static crear(nombre) {
        return new Animal(nombre, 0);
    }
}

class Perro extends Animal {
    constructor(nombre, raza) {
        super(nombre, 0);
        this.raza = raza;
    }

    saludar() {
        return `${super.saludar()}, soy un ${this.raza}`;
    }
}

const rex = new Perro("Rex", "Labrador");
console.log(rex.saludar());

Manejo de Errores

try {
    const resultado = operacionRiesgosa();
    if (!resultado) throw new Error("Resultado nulo");
} catch (error) {
    console.log("Error capturado:", error.message);
} finally {
    console.log("Siempre se ejecuta");
}
finally es opcional. catch tambien es opcional si hay finally.

Modulos (import / export)

Los tokens import, export, from, as, async, await, yield son reconocidos por el lexer. La semantica completa de modulos no esta implementada en la fase de parsing, pero los tokens se producen correctamente.

import { funcion } from './modulo';
export const PI = 3.14;
export default function principal() { ... }

Ejemplos

Programa completo

class Pila {
    constructor() {
        this.elementos = [];
    }

    apilar(item) {
        this.elementos.push(item);
    }

    desapilar() {
        if (this.elementos.length === 0) throw new Error("Pila vacia");
        return this.elementos.pop();
    }

    get tamano() {
        return this.elementos.length;
    }
}

const pila = new Pila();
pila.apilar(1);
pila.apilar(2);
pila.apilar(3);

for (const val of [10, 20, 30]) {
    pila.apilar(val);
}

while (pila.tamano > 0) {
    console.log(pila.desapilar());
}
